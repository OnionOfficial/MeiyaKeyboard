# **这是一个为芽宝打造的键盘**
*默认快捷键为：*

  Esc+Fn=~·

  1! +Fn=F1

  2@ +Fn=F2

  3# +Fn=F3

  4$ +Fn=F4

  5% +Fn=F5

  6^ +Fn=F6

  7& +Fn=F7

  8* +fn=F8

  9( +Fn=F9  
  

  0) +Fn=F10

  -_ +Fn=F11

  =+ +Fn=F12

  [{ +Fn=亮度+

  ]} +Fn=亮度-

  Fn +WASD=方向键

可自定义所有按键

软件可在[Releases](https://github.com/OnionOfficial/MeiyaKeyboard/releases)页面找到
